# webpack5 + ts + react 17.x + antd 4.x + dva 项目模板

## 介绍

webpack5 + ts + react 17.x + antd 4.x + dva 项目模板

- TODO: 后续会实现 cli, 一键生成项目

## 环境

## bash

```bash
  npm install / yarn

  npm run dev             # 启动

  npm run stylelint       # less 检查

  npm run stylelint:fix   # less fix

  npm run lint            # ts(x) 检查

  npm run lint:fix        # ts(x) fix

  npm run build           # 构建
```
